## 2. ビジョンの再確認：なぜ¥1,000億なのか

### 2.1 gemini.mdの核心命題

> 「ツールベンダー」から「B2Bグロースプラットフォーム」へ進化し、M&Aとデータ活用によって10年で年商¥1,000億を目指す。

この命題の本質は「Linear Growthの否定」だ。年商¥7億からの積み上げでは¥1,000億に30年以上かかる。非連続な成長（Jカーブ）を意図的に設計する必要がある。

### 2.2 市場定義の転換

| 現在の市場 | 目標の市場 | 倍率 |
|-----------|-----------|------|
| ウェビナーツール市場 ¥200億 | B2Bマーケティング/広告市場 ¥2兆 | 100倍 |

この転換は「既存事業の延長」ではない。**事業の定義そのものを変える。** 配信（Delivery）の前工程（Content Production）と後工程（Data Utilization / Audience Monetization）を垂直統合し、「配信ツール」ではなく「B2Bエンゲージメント経済圏」を構築する。

### 2.3 成長エンジン："AI Supply & Media Demand"

```
Supply（制作）: AI × Content Generation
  → 顧客が「指示一つ」で動画・資料・多言語コンテンツを制作
  → コンテンツ供給コストを1/10以下に

Demand（集客）: Open Nexpro Media Platform
  → 顧客許諾コンテンツを公開メディアに集積
  → 検索流入 → 新規リード → 顧客に還元

Data（循環）: Engagement Intelligence
  → 視聴データ蓄積 → AIレコメンド精度向上
  → ロックイン効果（使うほど離れられない）
```

**これがフライホイールだ。** Supply → Demand → Data → 更に良いSupply → 更に多いDemand → 更に深いData……と自己強化ループが回る。

## 3. Phase 1: Category King（〜¥20億）— 3つの資産を仕込む

### 3.1 Phase 1の再定義

Phase 1の目標は「国内No.1ウェビナーSaaSになること」**ではない。** それは手段であって目的ではない。

**Phase 1の真の目的は、Phase 2を可能にする「3つの資産」を構築すること。**

```
資産①: Tech OS     — 買収先にインストールする技術基盤
資産②: Open Nexpro — メディアモデルの小規模実証
資産③: Equity Story — SaaS+メディアのハイブリッド指標 → 資金調達
```

これら3つが揃って初めて、Phase 2のM&Aが「夢物語」ではなく「実行可能な戦略」になる。

### 3.2 資産①: Tech OS — 「買収先にインストールする技術基盤」

Phase 1で構築するプロダクト機能は、「SaaS顧客向けの新機能」であると同時に、「Phase 2で買収する企業に載せるオペレーティングシステム」として設計する。

#### Tech OSの構成要素

| コンポーネント | Phase 1での用途（SaaS） | Phase 2での用途（買収先DX） |
|--------------|----------------------|--------------------------|
| **エンゲージメントスコアリング** | 顧客のウェビナー視聴者をスコアリング→SF連携 | 買収したメディアの読者/来場者をスコアリング |
| **AI Content Generation** | ウェビナー→ブログ/クリップ/SNS自動生成 | 買収したメディアの既存コンテンツを動画/AI化 |
| **Dual-ID Architecture** | Private Lead（顧客資産）+ Public User（自社資産） | 買収先の会員DBをNexpro IDに統合 |
| **Salesforce連携基盤** | Agentforce × エンゲージメントデータ | 買収先の顧客に「商談化ROI」を可視化して提供 |
| **メディアサイトビルダー** | 顧客のオウンドメディア構築 | 買収したメディアのデジタル基盤をNexproに移行 |
| **API-first Architecture** | 外部ツール連携 | 買収先の既存システムとの統合を低コストで実現 |

**設計原則: すべての機能を「マルチテナント × マルチメディア」で構築する。**

Phase 1では「ウェビナー」という単一メディアタイプで動くが、アーキテクチャは「展示会」「業界メディア」「カンファレンス」「オンラインセミナー」のいずれにも適用可能な汎用性を持たせる。Phase 2で買収した企業のコンテンツを載せ替える際の移行コストを最小化するためだ。

### 3.3 資産②: Open Nexpro — 「メディアモデルの小規模実証」

gemini.mdの「Open Nexpro（公開メディア）」構想を、Phase 1の段階で**PoC規模で走らせる。** これはPhase 2のM&Aに対する最大の説得材料になる。

#### Open Nexproの設計

```
【構造】
既存顧客のウェビナーコンテンツ（許諾済み）
  ↓ AI Content Generationで変換
  ├── ショートクリップ（3-5分）
  ├── ブログ記事
  ├── FAQ/ナレッジベース
  └── ポッドキャスト
  ↓
Open Nexpro（公開メディアサイト）
  ↓ SEO/SNS/メルマガで集客
  ↓
来訪者
  ├── 匿名閲覧 → コンテンツ消費 → PV/広告収益
  └── Nexpro ID登録 → リード化 → 元の顧客に還元
```

#### Open NexproのKPI（Phase 1終了時 = M24目標）

| 指標 | 目標値 | 意味 |
|------|--------|------|
| 月間UU | 10万以上 | メディアとしてのトラフィック実証 |
| Nexpro ID登録数 | 5万以上 | Dual-IDのネットワーク効果の種 |
| コンテンツ→リード転換率 | 3%以上 | メディア→リードのビジネスモデル実証 |
| AI生成コンテンツ月間本数 | 1,000本以上 | Supply側のスケーラビリティ実証 |
| メディア起点の売上 | ¥50M/年以上 | 非SaaS収益の存在証明 |

**なぜこれが重要か:** Phase 2で業界メディアを買収する際、「ネクプロはメディアを運営した実績がある」「メディアからリードを生成してSaaS顧客に還元するモデルが実証済み」と言えるかどうかが、買収の成否を分ける。実績ゼロのSaaS企業がメディアを買っても、統合のノウハウがない。

### 3.4 資産③: Equity Story — 「SaaS+メディアのハイブリッド指標」

Phase 2のM&Aには資金が要る。資金調達（シリーズB/C）の成否は、Equity Story（投資家に語る成長ストーリー）の説得力で決まる。

#### Phase 1終了時に語れるEquity Story

```
【ストーリー】
「ネクプロは単なるウェビナーSaaSではない。
 SaaS収益（ARR ¥15-20億）+ メディア収益（Open Nexpro）+
 データ収益（エンゲージメントスコア）の3層収益モデルを持つ、
 日本唯一のB2Bエンゲージメントプラットフォームである。

 この"Tech OS"を、日本の旧来型B2Bメディアに水平展開する。
 日本のB2Bメディア/展示会市場は¥5,000億規模だが、
 デジタル化率は極めて低い。ネクプロはこの市場を
 テクノロジーで再定義する"日本版Cvent"になる。」
```

#### バリュエーションの論理

| シナリオ | Phase 1終了時の姿 | 想定PSR | 時価総額 | 調達可能額（20%希薄化） |
|---------|-------------------|---------|----------|----------------------|
| SaaS単体 | ARR ¥15-20億 | 8-12x | ¥120-240億 | ¥24-48億 |
| SaaS+メディア | ARR ¥15億 + メディア ¥3億 | 10-15x | ¥180-270億 | ¥36-54億 |
| プラットフォーム | 上記 + データ収益 + 高成長率 | 15-20x | ¥270-360億 | ¥54-72億 |

**ポイント:** 「SaaS単体」と「プラットフォーム」ではPSRが2倍近く変わる。Open Nexproとデータ収益は、売上寄与以上に**バリュエーション向上効果**が大きい。これが調達額の差となり、Phase 2のM&A原資の差になる。

## 4. Phase 1→2 ブリッジ：なぜM&Aか、何を買うか、どう変えるか

### 4.1 なぜオーガニック成長ではなくM&Aか

**オーガニック成長の限界を数字で示す。**

| 成長パス | 前提 | ¥100億到達時期 | 問題 |
|---------|------|---------------|------|
| SaaS単体オーガニック | ARR +25%/年 | 2040年頃（14年後） | 市場が待ってくれない |
| SaaS + 新収益オーガニック | +40%/年（計画値） | 2035年頃（9年後） | FY27計画の+52%が持続する前提 |
| SaaS + M&A | 3-4社買収 × Tech OS展開 | 2032年頃（6年後） | 実行リスクはあるが到達可能 |

**M&Aは「成長を買う」のではない。「オーディエンスを買い、テクノロジーで収益モデルを転換する」のだ。**

### 4.2 なぜ日本のB2Bメディア/展示会社が狙い目か

#### 構造的な買収機会

| 特性 | 詳細 | ネクプロにとっての意味 |
|------|------|---------------------|
| **バリュエーションが低い** | 伝統的メディアは収益横ばい〜逓減。売上2-4x（SaaSの8-15xの1/3以下）で取得可能 | 少ない資金で大きなオーディエンスを獲得 |
| **事業承継問題** | 業界紙・展示会社のオーナーは60-70代が多い。後継者不在率は全業種平均65%超 | 「売りたいが買い手がいない」状態 = 買い手有利 |
| **オーディエンスは価値がある** | 製造業・医療・金融・ITなど特定業種の「読者/来場者」リストを保有 | ネクプロのSaaS顧客と同じ業種のオーディエンス |
| **テクノロジーが弱い** | 紙→PDFレベルのDX。データ活用は皆無に近い | Tech OSの導入余地が極めて大きい |
| **コンテンツ資産がある** | 数十年分の業界専門コンテンツ（記事、講演録、技術資料） | AI Content Generationで動画化・再資産化可能 |

#### 買収候補の類型

| 類型 | 例（一般論） | 売上規模 | 保有するもの |
|------|-----------|---------|------------|
| **業界専門メディア** | 製造業向け技術メディア、医療専門誌 | ¥3-10億 | 読者リスト、専門コンテンツ、業界ネットワーク |
| **展示会運営会社** | B2B展示会・カンファレンス主催社 | ¥5-30億 | 来場者データ、出展企業ネットワーク、会場運営ノウハウ |
| **業界団体メディア** | 業界団体の出版・イベント部門 | ¥1-5億 | 会員リスト、業界標準へのアクセス |
| **企業研修会社** | セミナー/研修を運営する中小企業 | ¥2-8億 | 受講者リスト、講師ネットワーク、教材 |

### 4.3 買収後の価値転換モデル（Unit Economics）

**1社あたりの買収→転換→収益化のシミュレーション。**

#### ケーススタディ: 製造業向け展示会運営会社の買収

```
【買収前】
  売上: ¥8億（出展料¥5億 + 広告掲載¥2億 + その他¥1億）
  営業利益率: 8%（¥6,400万）
  来場者データ: 5万人/年（名刺情報レベル、デジタル化されていない）
  コンテンツ: 過去10年分の講演録・技術資料（PDF/紙）
  バリュエーション: 売上3x = ¥24億

【Phase 2-Year 1: Tech OS導入】
  ①ハイブリッド化: 展示会をオン/オフ併設 → オンライン参加者+3万人
  ②Nexpro ID発行: 来場者/視聴者にID発行 → 8万人のデジタルプロファイル
  ③コンテンツDX: 講演→AI動画化、資料→オンデマンドコンテンツ化
  ④エンゲージメントスコア導入: 来場者の関心領域・熱量をスコアリング

  売上: ¥8億 → ¥9.5億（オンラインチケット+デジタル広告追加）
  コスト増: ¥5,000万（Tech OS運用+人件費）
  営業利益率: 8% → 12%

【Phase 2-Year 2: データ収益化】
  ⑤インテントデータ販売: スコアリングされた来場者データを出展企業に提供
  ⑥リード課金モデル: 出展料の一部を「リード獲得課金」に転換
  ⑦365日メディア化: 展示会を年1回イベント → 365日オンラインメディアに
  ⑧SF連携: 出展企業のSalesforceに商談化データを直接フィード

  売上: ¥9.5億 → ¥14億（データ/リード課金¥3億 + 365日メディア¥1.5億）
  営業利益率: 12% → 22%（テック化で粗利改善）

【Phase 2-Year 3: プラットフォーム効果】
  ⑨クロスメディア: 他の買収メディアとのオーディエンス相互送客
  ⑩Nexpro IDネットワーク効果: ID保有者が増えるほどデータの価値が上がる

  売上: ¥14億 → ¥18億
  営業利益率: 22% → 28%
```

**このフライホイールの自己強化メカニズム:**

1. 買収したメディアのコンテンツ + SaaS顧客のウェビナーコンテンツ → AI Generationで変換 → Open Nexproに集積

5. SaaS顧客が増える → ウェビナーコンテンツが増える → 1に戻る

**Moat（堀）の深化:** 回すほどデータが溜まり、データが溜まるほど離脱しにくくなる。競合が「配信ツール」を作っても、このデータネットワーク効果は模倣できない。

### 6.2 Phase 3の方向性

gemini.mdの「デジタル商社」構想を具体化する。

**Phase 3には2つの方向性がある:**

#### 方向性A: グローバルB2Bメディアロールアップ

Phase 2で日本で実証した「メディア買収 → Tech OS導入 → データ収益化」モデルを、アジア→欧米に展開する。

- アジアのB2B展示会/メディア市場はデジタル化が遅れている
- 日本で磨いたTech OSを横展開
- 「日本品質のテック × 現地のオーディエンス」

#### 方向性B: B2Bクロスボーダーコマース

日本のB2B製品を、Nexpro IDの海外ネットワーク経由で海外のバイヤーに紹介する「デジタル商社」機能。

- 日本企業は優れたB2B製品を持つが、海外マーケティングが弱い
- Nexpro Media Networkの海外オーディエンスに、動画/AIコンテンツで紹介
- 商社のマッチング機能を、テクノロジーで再定義

**Phase 3の設計はPhase 2の結果を見て決定する。** 現時点では方向性を示すに留め、Phase 2のGate Reviewで具体化する。

compass_artifactが指摘するように、**AIエージェントが配信機能をコモディティ化する流れは不可逆**だ。

```
何もしない場合のシナリオ:
  FY24: ¥5.1億（現在）
  FY25: ¥5.5億（微増）
  FY26: ¥5.0億（Zoom AI Companionに顧客流出開始）

  FY28: ¥3.0億（コモディティ化の帰結）
  → 5年以内に事業の存続が問われる
```

**「何もしないこと」が最もリスクの高い選択肢である。**

## 11. 結論：3つの命題

### 命題1: Phase 1は「SaaSの成長」ではなく「M&Aプラットフォームの構築」

エンゲージメントスコア、AI Content Generation、Dual-ID、API-first——これらは「SaaS機能」ではなく、**買収した旧来型メディアにインストールする「オペレーティングシステム」の構成部品**だ。Phase 1のすべてのプロダクト判断は、「これはPhase 2で横展開できるか？」というフィルターを通す。

### 命題2: Phase 2は「メディア買収」ではなく「オーディエンスのテクノロジー転換」

旧来型メディアの価値は「オーディエンス」にある。しかし、そのオーディエンスは紙・名刺・来場者リストという形で死蔵されている。ネクプロのTech OSをインストールすることで、「死蔵されたオーディエンス」を「リアルタイムのインテントデータ」に変換する。**これがM&Aの本質であり、「安く買って高く売る」というアービトラージではない。**

### 命題3: ¥1,000億は「大きな数字」ではなく「市場の大きさから逆算した到達点」

B2Bマーケティング/広告市場は¥2兆。その5%を取れば¥1,000億。5%は支配的シェアではなく、十分に現実的な目標だ。問題は「市場が小さい」ことではなく、「その市場にアクセスする手段を持っていない」ことだ。Phase 1-3は、そのアクセス手段を段階的に構築するプロセスである。

# ネクプロ 全社戦略提案：ウェビナーツールからB2Bエンゲージメント・インテリジェンス基盤へ

**対象期間：2026-2028（0-36ヶ月）**
**会議目的：経営層・マネージャーMTGにおける3つの意思決定**

---

## 1. エグゼクティブサマリー

**スライドタイトル：** 結論 — 「配信ツール」に留まれば死ぬ。「データ基盤」に転換すれば勝てる。

**キーメッセージ：**
ネクプロは今、「ウェビナーツール」から「B2Bエンゲージメント・インテリジェンス基盤」への不可逆的転換を決断すべきタイミングにある。

**ボディ（図解イメージ）：** 2×2マトリクス — 縦軸「市場成長性」横軸「自社競争力」で4象限を配置。現在地（ウェビナーツール＝低成長×中競争力）→ 目標地（エンゲージメント基盤＝高成長×高競争力）への矢印。

### サマリー要点

| 項目 | 現状 | 目標（36ヶ月後） |
|------|------|------------------|
| 事業定義 | ウェビナー配信ツール | B2Bエンゲージメント・インテリジェンス基盤 |
| 売上規模 | ¥512M（FY24実績） | ¥1,382M（FY27計画） |
| MRR | ¥287M/年（FY24） | ¥526M/年（FY27） |
| ARPA（長期） | ¥150K/月 | ¥204K/月 |
| 月次解約率（長期） | 1.7%（FY24実績） | 1.0%（目標） |
| 新収益柱 | なし | コンパウンド¥51M + 営業DX¥327M |
| 顧客数（長期PF） | 167社 | 210社 |

### 本日の意思決定事項（3つ）

1. **資源配分の決定：** プロダクト45% / GTM35% / 組織20%（仮説）の投資配分承認
2. **ポジショニング転換の承認：** 「ウェビナーツール」→「B2Bエンゲージメント・インテリジェンス基盤」への事業再定義
3. **組織再編の承認：** PMM/RevOps機能新設、KPIオーナー設計、90日実行計画

**根拠：**
- AIエージェント時代にUI中心SaaSの中間層は圧縮される（SaaSpocalypse：SaaS株$2,850億消失）
- 日本ウェビナーSaaSはAI統合で12-18ヶ月遅れ → 先行者優位の時間的猶予あり
- Agentforce統合は国内最先行だが「序章」にすぎない

**反証リスク：**
- 30名体制で基盤転換と既存事業維持の両立は困難な可能性
- 新収益柱（営業DX ¥327M/FY27）は未実証の仮説値

## 2. 外部環境（AIエージェント時代の産業変化）

**スライドタイトル：** SaaS中間層の圧縮 — AIエージェントが変えるソフトウェア産業の構造

**キーメッセージ：**
AIエージェントはSaaSの「UIレイヤー」を不要にするが、「データレイヤー」の価値は増大する。

**ボディ（図解イメージ）：** 3層構造図 — 上層「AIエージェント（実行層）」/ 中間「SaaS UI（圧縮対象）」/ 下層「システム・オブ・レコード（データ層＝価値増大）」

### 外部環境ファクト

| ファクト | 数値 | ソース |
|----------|------|--------|
| SaaSpocalypse株価消失 | $2,850億（約42兆円） | 2026年2月、Claude Coworkプラグイン発表後 |
| バーティカルSaaS年初来下落 | −43% | Tomasz Tunguz分析（2026.2.6） |
| ワークフローSaaS下落 | −39% | 同上 |
| AIエージェントプロジェクト中止予測 | 40%以上（2027年末まで） | Gartner 2026.2 |
| シートベース→成果ベース課金移行 | 70%（2028年まで） | IDC予測 |

### 日本市場の構造的特性

**参入障壁（時間稼ぎ要因）：**
- SaaS浸透率：日本4%（米国15-18%）
- 商習慣の壁（年末調整・インボイス制度・J-GAAP）
- 言語の壁（日本語バイヤーの72%が母国語選好）
- デジタル人材：2026年までに230万人不足

**ただし：** これらは「時間稼ぎ」であり永続的防壁ではない。ソフトウェア支出10兆円に対しヒト支出300兆円がAI代替対象。

**ネクプロへの示唆：**
- ウェビナー「配信機能」は battlegrounds（AIが蚕食する領域）
- ウェビナー「エンゲージメントデータ」は gold mines（AI活用で価値増大する領域）
- **判断基準：データを持つ側が勝つ。UIを提供するだけの側は負ける。**

**反証リスク：**
- 日本市場の保守性がAI転換を遅延させ、投資回収が想定より長期化する可能性

1. **FY24売上成長率 +3.1%** — FY23の+19%から急減速。MRR成長に依存しすぎ

4. **30名体制の限界** — 開発・CS・営業を同時に強化する余力が不足

**反証リスク：**
- FY24の成長鈍化が一時的要因（大型案件の有無等）の可能性。構造的問題か検証必要
- 新収益柱（営業DX等）のFY26-27計画は急角度の成長カーブ。実行リスク大

## 4. 戦略的SWOT分析

**ボディ（図解イメージ）：** SWOT 2×2マトリクス + 下段にクロスSWOT戦略マトリクス

### Strength（強み）

## 8. 推奨戦略とその根拠

**スライドタイトル：** 推奨 — 「竹」で6ヶ月の実績を作り、「松」へ段階移行

**キーメッセージ：**
「全力投球ではなく、実績で信頼を獲得しながらギアを上げる」アプローチ。

### 推奨理由（3つのロジック）

**1. キャッシュ耐性の確保**
- 30名・調達¥7.5億の企業が全方位投資するのは自殺行為
- 竹案なら投資増分+30-50%で既存事業のキャッシュフローを維持可能

**2. 「小さく証明して大きく張る」パターン**
- エンゲージメントスコアMVPを6ヶ月で構築 → 5社で成果実証 → 本格投資判断
- 先行事例：Sansan「DX→AX」転換も段階的に実行

**3. 組織能力の段階的構築**
- いきなりPMM/RevOpsを正式新設しても機能しない
- 兼務→成果確認→専任化のステップが現実的

### やること / やらないこと

| PMM兼務設置 | 大規模採用（初期） |
| 解約率改善（1.7%→1.0%） | 新規大量獲得（質を優先） |

## 想定Q&A（厳しめの指摘への回答）

**Q1: 30名でこれだけの施策を同時に実行できるのか？**
A1: できない。だから「竹案」で優先順位を絞っている。最優先は①エンゲージメントスコアMVP ②解約率改善 ③業種別パッケージの3つ。他は段階的に実行する。全部同時にやろうとすれば失敗する。Gate Reviewで判断し、リソースが確保できた時点で松案に移行する。

**Q2: 「エンゲージメント・インテリジェンス基盤」と言うが、結局ウェビナーツールではないのか？**
A2: 現時点ではその通り。だが、ON24が参加者あたり40-50のデータポイントでインテントスコアを構築し、AutodeskがパイプラインROI 12倍を達成している事実がある。ネクプロが持つ400社超のウェビナーデータを構造化すれば、同等の価値を日本市場で提供できる。ウェビナーは「データ収集装置」であり、価値の源泉はデータの方にある。

**Q3: 営業DXの売上¥327M（FY27）は絵に描いた餅ではないか？**
A3: 正直に言えば実行リスクは高い。FY25の¥33.4Mがまず達成できるかが最初のリトマス試験。6ヶ月後のGate Reviewで¥15M/半期を下回った場合は、計画を下方修正するか事業ピボットを検討する。計画に盲目的にコミットするのではなく、早期のファクトベース判断を仕組みとして組み込んでいる。

**Q4: Zoom/Webex/TeamsがAI機能を搭載してきたら勝てないのでは？**
A4: 配信機能で勝負すれば確実に負ける。だからこそ「配信ツール」から脱却する。Zoom等は汎用ツールであり、日本B2Bマーケティングファネル（申込→配信→分析→CRM連携→ナーチャー）に最適化する動機がない。ネクプロの勝ち筋は「日本のB2B商習慣に最適化されたエンゲージメントデータ×Salesforce連携」という、彼らが手を出さないニッチ。

**Q5: Salesforce依存は危険ではないか？中期でHubSpot/Marketo連携を入れるなら最初からやるべきでは？**
A5: 30名体制で3つのCRM/MA統合を同時開発するのは自殺行為。Salesforce連携がネクプロの最大の差別化資産であり、これを深化させるのが先。HubSpot/Marketoは中期（6-18ヶ月）でAPI-firstアーキテクチャが完成した後に対応する。「広く浅く」ではなく「狭く深く→段階拡大」。

**Q6: 解約率1.0%目標は現実的か？現在1.7%だが。**
A6: 長期PFの月次解約率はFY22の3.6%→FY23の2.3%→FY24の1.7%と改善トレンドにある。ヘルススコア導入とオンボーディング標準化で0.7pt改善は達成可能と見ている。ただし、解約理由の分析が不十分な現状では保守的に見る必要もある。失注理由TOP10のデータ収集を即時開始する。

**Q7: PMM/RevOpsを兼務で始めて本当に機能するのか？**
A7: 正式採用から始めると人材のミスマッチリスクがある。兼務で3ヶ月回して「何をすべきか」の解像度を上げた後に、専任化 or 採用判断する方が合理的。兼務の弱点は「本業との優先度競合」なので、CEO直轄にして工数の20%を確保する運用ルールを設定する。

**Q8: FY26-27の売上計画は攻めすぎではないか？**
A8: FY26 ¥911M（+40%）、FY27 ¥1,382M（+52%）は確かに攻めた計画。内訳を見ると、MRR成長は年+25-26%で安定的だが、オプションサービスと新規事業（営業DX+コンパウンド）が大きく伸びる前提。特に営業DXの¥121M（FY26）→¥327M（FY27）は急角度。これが未達の場合のダウンサイドシナリオも準備しておく必要がある。竹案の¥930Mが現実的なベースケース。

**Q9: 競合のbiziblやFanGrowthとの差別化は本当にあるのか？**
A9: biziblは低〜中価格帯のウェビナーツール、FanGrowthは集客支援・成果連動型。いずれもデータ分析やAI機能は未発達。ネクプロとの差別化は①Salesforce深度統合 ②エンゲージメントデータの構造化 ③Agentforce対応 の3点。ただし、これらは「今の差別化」であり、6-12ヶ月で他社が追随する可能性はある。だからこそ今すぐ動く必要がある。

**Q10: そもそもウェビナー市場自体が縮小しないか？対面イベント回帰の流れもある。**
A10: グローバルデータでは企業あたりウェビナー数が年13回→47回に急増（Goldcast調査）。対面イベントとの共存であり、ウェビナー自体の需要減退は見られない。むしろハイブリッド化により、対面+オンラインの両方でデータ収集する需要が増大。ただし、国内データでこの傾向を裏付ける定量情報が不足しているため、顧客アンケート等で検証すべき。

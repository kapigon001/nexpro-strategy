## 2. 外部環境分析 — AIエージェント時代のSaaS構造変化

### 2-1. SaaS中間層の圧縮

> AIエージェントはUIレイヤーを不要にするが、データレイヤーの価値は増大する

**SaaS産業の3層構造変化:**

```
┌─────────────────────────────────┐
│  AIエージェント層（実行）  ↑拡大  │
├─────────────────────────────────┤
│  SaaS UIレイヤー（圧縮対象）←→圧縮│
├─────────────────────────────────┤
│  データ基盤層（価値増大）   ↑拡大  │
└─────────────────────────────────┘
```

| ファクト | 数値 |
|---------|------|
| SaaS株価消失（SaaSpocalypse） | $2,850億（約42兆円） |
| バーティカルSaaS下落 | -43% |
| ワークフローSaaS下落 | -39% |
| AIエージェントPJ中止予測（2027年末） | 40%+ |
| 日本SaaS浸透率 | 4%（米国15-18%） |

### 2-2. 日本市場の構造的特性

> SaaS浸透率4%の日本は12-18ヶ月の先行者優位。今動けば間に合う

**日本法人向け動画配信市場:**

| 年度 | 市場規模 |
|------|---------|
| 2023 | ¥513B |
| 2024 | ¥635B |
| 2025（推定） | ¥780B |
| 2026（推定） | ¥950B |
| 2027（推定） | ¥1,200B |
| 2028 | ¥1,529B |

**日本市場の4つの参入障壁（時間稼ぎ要因）:**

1. **SaaS浸透率 4%**（米国15-18%）— 市場が未成熟で先行者優位を取れる
2. **日本語選好 72%のバイヤー** — グローバル勢に対する言語の壁
3. **デジタル人材不足 230万人** — AI導入の実行リソースが不足
4. **商習慣の壁（稟議・根回し文化）** — 海外SaaSの浸透を遅延させる構造的要因

## 4. SWOT分析

> 「データモート」が唯一の持続的優位性

### 4-1. SWOT

#### S（強み）
1. **ITreview 13期連続受賞の製品品質** — ウェビナー/動画配信カテゴリーで継続的な高評価
2. **Salesforce Agentforce国内最先行** — 日本のウェビナーツールで最もSF連携が深い
3. **400-500社エンタープライズ顧客基盤** — 長期PFアカウント167社（FY24）の安定基盤
4. **日本語運用ノウハウ・伴走支援** — 国内企業の商習慣を理解した支援体制
5. **擬似ライブ配信・メディアサイト機能** — 競合にない独自機能で差別化

#### W（弱み）
1. **30名の少数精鋭体制** — 同時並行施策の実行に限界
2. **総調達額¥7.5億（対Zoom/ON24比）** — 投資余力が桁違いに少ない
3. **ブランド認知度（Zoom,V-CUBEに劣後）** — トップオブマインドでの想起率が低い
4. **インテントデータ未整備（ON24は40-50データポイント/人）** — データ分析の深度で劣後
5. **PMM/RevOps機能の不在** — 市場戦略・ファネル最適化の専門機能がない

#### O（機会）
1. **国内AI統合12-18ヶ月遅れ（先行者優位）** — 日本のウェビナーSaaSでAI統合した企業はまだない
2. **ON24のCvent買収→日本市場優先度低下** — 最大競合の日本注力が下がる猶予
3. **3rdパーティCookie廃止→1stパーティデータ価値増** — ウェビナーデータ=貴重な1stパーティデータ
4. **日本B2B DX余地（SaaS浸透率4%）** — 市場拡大の余地が大きい
5. **企業あたりウェビナー数急増（13→47回/年）** — 需要の構造的拡大

#### T（脅威）
1. **Zoom/Webex/TeamsのAIエージェント搭載** — グローバル勢がAI機能を急速に拡充
2. **AIによる配信機能コモディティ化** — 基本機能の差別化が困難に
3. **Cvent大型統合（ON24 ¥400億+Goldcast ¥300億）** — エンゲージメント領域の巨大プレイヤー出現
4. **価格競争激化（bizibl/Cocripo低価格攻勢）** — 国内の低価格帯の攻勢
5. **汎用AIで差別化の希薄化リスク** — 独自機能がAIで代替される可能性

### 4-2. クロスSWOT戦略示唆

> S×Oの「データモート構築」とW×Tの「最大リスク回避」が戦略の両輪

| | 機会（O） | 脅威（T） |
|--|----------|----------|
| **強み（S）** | SF統合×AI先行で「データモート」構築 → **エンゲージメントスコアMVP** | 日本適合性×伴走支援でグローバル勢と差別化 → **業種別パッケージ** |
| **弱み（W）** | 少数精鋭×AI遅れの隙間でニッチ勝ち → **SF顧客に選択と集中** | **【最大リスク】** 資金・人材不足×コモディティ化 → **何もしなければ3年以内に地位喪失** |

## 8. 推奨戦略

> 全力投球ではなく、実績で信頼を獲得しながらギアを上げる

### 推奨の3つの論拠

1. **キャッシュ耐性:** 30名・¥7.5億で全方位投資は自殺行為
2. **小さく証明:** MVP 6ヶ月→5社実証→本格投資
3. **段階構築:** 兼務→成果確認→専任化

### やること / やらないこと

| やること ✓ | やらないこと ✗ |
|-----------|-------------|
| エンゲージメントスコアMVP | 汎用AIチャットボット開発 |
| Salesforce連携深化 | HubSpot/Marketo（中期以降） |
| 業種別ソリューション化 | 低価格競争 |
| CS→プロフィットセンター化 | 機能の無秩序追加 |
| 解約率改善（1.7%→1.0%） | 新規大量獲得（質を優先） |

### Gate Review基準（6ヶ月後）

| KPI | Gate基準 | 達成時 | 未達時 |
|-----|---------|--------|--------|
| スコアMVP導入社数 | 5社以上 | 松案移行 | 竹延長・修正 |
| 月次解約率 | ≤1.3% | 松案移行 | CS施策見直し |
| ARPA（長期） | ≥¥160K | 松案移行 | 価格体系再検討 |

## 想定Q&A（厳しい質問10選）

### Q1: 30名で本当に同時実行できるのか？
**A:** 竹案は「選択と集中」が前提。全方位ではなく5課題に絞る。兼務→専任の段階アプローチで現実的に対応。Gate Reviewで実行可能性を検証。

### Q2: 結局ウェビナーツールの延長ではないか？
**A:** エンゲージメントスコアは「配信の先」のデータ活用。Salesforce商談データとの接続により、マーケティングROI可視化という新カテゴリーを定義。ON24のIEP（Intelligent Engagement Platform）モデルが先例。

### Q3: 営業DX ¥327Mは絵に描いた餅では？
**A:** FY25の¥33Mが実績のシード。Gate Review基準（¥15M/半期）を設定し、未達時はピボット。段階的に投資を増やす設計。

### Q4: Zoom/Teams/WebexのAI搭載で勝てるのか？
**A:** グローバル勢の日本適合は構造的に弱い（日本語選好72%、商習慣の壁）。12-18ヶ月の猶予期間にデータモートを構築すれば、機能コモディティ化の影響を受けにくい。

### Q5: Salesforce依存は危険では？
**A:** 依存ではなくエコシステム活用。HubSpot/Marketo連携は中期（12-18M）で展開予定。Agentforce対応は国内最先行であり、先行者優位を活かすべき。

### Q6: 解約率1.0%は現実的か？
**A:** FY22:3.6%→FY23:2.3%→FY24:1.7%と改善トレンドにある。ヘルススコア+オンボーディング標準化+戦略アカウント管理の3施策で到達可能。ただしGate基準は1.3%（バッファ付き）。

### Q7: PMM/RevOpsを兼務で機能するか？
**A:** 最初から専任は30名体制では非現実的。兼務で3-6ヶ月試行し、成果が出れば専任化（採用）。成果が出なければ打ち手を変える。Gate Reviewで判断。

### Q8: FY26-27計画は攻めすぎでは？
**A:** 竹案（推奨）のFY27は¥930M（仮説）であり、松案の¥1,382Mは達成条件付き。Gate Reviewで段階移行するため、コミットではなく「条件付き目標」。

### Q9: bizibl/FanGrowthとの差別化は？
**A:** 両社は「簡単・安い」がポジション。ネクプロは「データ深度×SF連携×伴走支援」で差別化。競合軸が異なるため、直接的な価格競争は回避可能。

### Q10: ウェビナー市場自体が縮小しないか？
**A:** 逆に拡大。企業あたりウェビナー数は13回→47回（3.6倍）に急増。市場は¥513B→¥1,529B（CAGR24.4%）。ただし「配信だけ」の市場は縮小し、「データ活用」市場が成長する構造変化。

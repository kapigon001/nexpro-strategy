# Strategy Presentation Generator Skill

**マッキンゼー・BCG流の「論理的アプローチ」で、意思決定のための戦略資料を自動生成するClaude Codeスキル**

このスキルは、単にテンプレートを埋めるのではなく、与えられたデータとゴールに基づいて「ストーリーライン（構成）」を動的に設計し、Pythonコード経由でPowerPointファイルを生成します。

---

## ✨ 特徴

1.  **Dynamic Storylining (動的構成)**
    * 画一的なテンプレートを使用しません。「予算承認」「新規事業提案」「進捗報告」など、ゴールに合わせて最適な章立て（Ghost Deck）を提案します。
2.  **Fact & Logic Driven**
    * 「なんとなく」の主張を排除します。提供されたデータソース（md, csv, json等）から事実を抽出し、ピラミッドストラクチャーに基づいて論理を構築します。
3.  **Code-First Generation**
    * 直接バイナリ（pptx）を生成するのではなく、**「スライド生成用のPythonスクリプト」を作成・実行**します。
    * これにより、後から色味やレイアウトを微調整したり、データを差し替えて再生成することが容易です。
4.  **Audience Centric**
    * 「誰に」「何を決めてもらうか」を最優先に設計されます。

---

## 🚀 使い方

### 1. 準備
分析対象となるドキュメントやデータを、プロジェクトフォルダ内に用意してください。
* **議事録/メモ**: `notes.md`, `meeting_log.txt`
* **定量データ**: `sales_data.csv`, `kpi_metrics.json`
* **既存資料**: `background.pdf` (テキスト抽出可能な場合)

### 2. 実行
Claude Code上で以下のように指示します。
